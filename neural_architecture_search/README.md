# Deep Bayes Optimization Neural Architecture Search (DBONAS)

## Installation

```
$ cd neural_architecture_search
$ pip install -e .
```

## Test

```
$ cd neural_architecture_search
$ python setup.py test
```

## Examples

```
$ cd neural_architecture_search/examples
$ python mnist.py
```
